Thin wrapper CLI to run `@toxuh/create-imp` via `npx imp`.

## Install / Usage

```
npx @toxuh/imp
```

This invokes `@toxuh/create-imp` and guides you through project scaffolding.

## Development

- Build: `npm run build`
- Test: `npm run test`
- Lint: `npm run lint`
